CRF-FAQ’s.

Please read the following points before proceeding with the CRF Entry.

1. All CRF changes except Nomination & Bank are PAN based.

2. The process is based on the KRA/ CKYC status of the client.

  • A. Active client with valid KRA/ CKYC- Choose the appropriate option in the module and proceed to pass the entry.

  • B. Active / Inactive clients without KRA/ CKYC- You will not be able to pass the entry till the KYC is updated. In order to update the client's KRA/ KYC, click “Next” on the module window to proceed.

  • C. Inactive client with valid KRA/ CKYC- Entry can be passed directly through the module. Financial’s updation can also be done through the module.

3. How to print Barcoded CRF forms ?

  • Option 1 – Print barcoded CRF forms with data

    • Fill in the required data in the CRF module. Then take the print out and obtain client’s signature.

  • Option 2 – Print barcoded CRF forms without data

    • All forms which need clients’ signature can be printed from this module. Select the required option from the main menu. You will be able to print the concerned form with basic client credentials.
